一个安全，开源，本地运行的全功能插件。
插件功能：
1. 自动实时翻译（谷歌翻译）。
2. 支持推特标签备注(点击✏备注)。
3. 支持推特重点高亮关注（点击⭐）。
类似小何 
@heyibinance
 推文显示效果。
4. 支持设置字体颜色大小，怎么喜欢怎么来。
5. 支持本地备份导入，其他家的备份也支持导入使用。
6. 支持CA检测，点击一键跳转到行情界面。
